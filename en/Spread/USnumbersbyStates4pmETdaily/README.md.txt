COVID19 Dataset :  (USnumbersbyStates4pmETdaily)
COVID19 US Numbers By States daily 4pm ET 

File naming convention:
   <dataset_name>-<YYYYMMDD>.CSV

Field description:
Date			: Date
State			: US State code 
Positive		: Number of poistive cases
Negative		: Number of negative cases	
Pending			: Number of pending cases
HospitalizedCurrently	: Number of people currently in hospital
HospitalizedCumulative	: Number of people in hospital (cumulative)
InICUcurrently		: Number of people currently in ICU
InICUcumulative		: Number of people in ICU (cumulative)
OnVentilatorCurrently	: Number of people currently on ventilator
OnVentilatorCumulative	: Number of people on ventilator (cumulative)
Recovered		: Number of people that have recovered
Deaths			: Number of dead
DataQualityGrade	: Data Quality Grade
LastUpdateET		: Last Update (ET)

Update frequency:
Once a day 4pm ET.

Data API:
https://covidtracking.com/api
https://covidtracking.com/api/v1/states/daily.json
