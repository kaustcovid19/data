COVID19 Dataset :  (FullListDailyCovid19TestsPerK)
COVID19 Daily total number of tests, or people tested (Per 1000 people)

File naming convention:
   <dataset_name>-<YYYYMMDD>.CSV

Field description:
Entity			: Country
Code			: Country Code
Date			: Date occured
DailyChangeInTotalTests	: Number of tests, or people tested (Per 1000 people)

Update frequency:
Daily

Note :
Note: There are substantial differences across countries in terms of the units, whether or not all labs are included, the extent to which negative and pending tests are included and other aspects

Data sources:
https://ourworldindata.org/grapher/full-list-daily-covid-19-tests-per-thousand
