COVID19 Dataset :  (FullListCumulativeTotalTestsPerK)
COVID19 The total number of tests performed or people tested so far Per 1000 people

File naming convention:
   <dataset_name>-<YYYYMMDD>.CSV

Field description:
Entity		: Country
Code		: Country Code
Date		: Date occured
TotalTestsPerK	: Cumulative Tests for COID-19 Per 1000 People

Update frequency:
Daily

Note: 
There are substantial differences across countries in terms of the units, whether or not all labs are included, the extent to which negative and pending tests are included and other aspects.

Data sources:
https://ourworldindata.org/grapher/full-list-cumulative-total-tests-per-thousand 
Original Source file : full-list-cumulative-total-tests-per-thousand.csv
