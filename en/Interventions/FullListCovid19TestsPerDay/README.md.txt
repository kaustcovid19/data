COVID19 Dataset :  (FullListCovid19TestsPerDay)
COVID19 Daily total number of tests, or people tested (Absolute number)

File naming convention:
   <dataset_name>-<YYYYMMDD>.CSV

Field description:
Entity			: Country
Code			: Country Code
Date			: Date occured
DailyChangeInTotalTests	: Number of tests, or people tested (Absolute number)

Update frequency:
Daily

Note :
Note: There are substantial differences across countries in terms of the units, whether or not all labs are included, the extent to which negative and pending tests are included and other aspects

Data sources:
https://ourworldindata.org/grapher/full-list-covid-19-tests-per-day
Original Source file : full-list-covid-19-tests-per-day.csv
