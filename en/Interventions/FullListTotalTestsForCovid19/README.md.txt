COVID19 Dataset :  (FullListTotalTestsForCovid19)
COVID19 The total number of tests performed or people tested so far (Absolute number)

File naming convention:
   <dataset_name>-<YYYYMMDD>.CSV

Field description:
Entity			: Country
Code			: Country Code
Date			: Date occured
TotalTests		: Number of tests, or people tested (Absolute number)

Update frequency:
Daily

Note :
Note: There are substantial differences across countries in terms of the units, whether or not all labs are included, the extent to which negative and pending tests are included and other aspects

Data sources:
https://ourworldindata.org/grapher/full-list-total-tests-for-covid-19
