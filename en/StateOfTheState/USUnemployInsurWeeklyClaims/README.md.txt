COVID19 Dataset : (USUnemployInsurWeeklyClaims)
COVID19  US Unemployment Insurance Weekly Claims:Seasonally Adjusted Data

File naming convention:
   <dataset_name>-<YYYYMMDD>.CSV

Field description:
WeekEnding	: Date of the week
Initial Claims	: $

Update frequency :
Weekly (but usually takes 12 days to update)

NOTE : By the time the next weekly update is recieved it is observed that Government performs some adjustments to the previous week figures too!

Data sources:
https://www.dol.gov/ui/data.pdf
https://oui.doleta.gov/unemploy/claims.asp
